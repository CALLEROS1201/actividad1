\# What is Markdown?



Markdown is a lightweight markup language used to format text easily.

It allows users to create structured documents using plain text syntax.



Markdown is commonly used for:

\- README files

\- Documentation

\- GitHub projects

\- Notes and blogs



It is popular because it is simple, readable, and converts easily into HTML.
